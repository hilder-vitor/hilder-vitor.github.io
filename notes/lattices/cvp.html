CVP

  • - Decisional, optimization, and search versions are (polynomial) equivalent.
  • - CVP is NP-complete (reduction from subset-sum problem).
  • - Given an oracle to $CVP_\gamma$, it is possible to solve $SVP_\gamma$ in polynomial time.
  • - For all $p$, approximation version with factor $\log^c (n)$ and $\ell_p$ norm is NP-Hard. [MC2002, chapter 3]
  • - For all $p$, approximate version in $\ell_p$ norm with factor $< (5/3) ^ {\frac 1 p}$ is NP-Hard. [MC2002, chapter 3]
  • - Approximate CVP with factors bigger than $O\left(\frac{ \sqrt{n} } {\log{n}} \right)$ is not expected to be NP-Hard. [MC2002, chapter 9]
